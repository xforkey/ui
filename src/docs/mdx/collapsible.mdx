## Usage

<CodeExampleStack>

```tsx

  Collapsible,
  CollapsibleContent,
  CollapsibleTrigger,
```

```tsx
<Collapsible>
  <CollapsibleTrigger>Can I use this in my project?</CollapsibleTrigger>
  <CollapsibleContent>
    Yes. Free to use for personal and commercial projects. No attribution
    required.
  </CollapsibleContent>
</Collapsible>
```
