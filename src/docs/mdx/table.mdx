## Usage

```tsx

  Table,
  TableBody,
  TableCaption,
  TableCell,
  TableFooter,
  TableHead,
  TableHeader,
  TableRow,
} from "@/components/ui/table"
```

```tsx
<Table>
  <TableCaption>A list of your recent invoices.</TableCaption>
  <TableHeader>
    <TableRow>
      <TableHead className="w-[100px]">Invoice</TableHead>
      <TableHead>Status</TableHead>
      <TableHead>Method</TableHead>
      <TableHead className="text-right">Amount</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell className="font-medium">INV001</TableCell>

  </TableBody>
  <TableFooter>

      <TableCell colSpan={3}>Total</TableCell>

</Table>
```

## Props

### Table

Extends `React.ComponentProps<"table">`

| Prop | Type | Default | Description |
| :-- | :-- | :-- | :-- |
| `bleed` | `boolean` | `false` | When `true`, the table will bleed to the edges of its container. |
| `dense` | `boolean` | `false` | When `true`, the table cells will have less padding. |
| `grid` | `boolean` | `false` | When `true`, the table will have grid lines. |
| `striped` | `boolean` | `false` | When `true`, the table rows will have alternating background colors. |

### TableHeader

Extends `React.ComponentProps<"thead">`

### TableBody

Extends `React.ComponentProps<"tbody">`

### TableFooter

Extends `React.ComponentProps<"tfoot">`

### TableRow

Extends `React.ComponentProps<"tr">`

### TableHead

Extends `React.ComponentProps<"th">`

### TableCell

Extends `React.ComponentProps<"td">`

### TableCaption

Extends `React.ComponentProps<"caption">`
